# Jivon Research Navigator

新的综合科研导航仓库，整合 ORCID 学术画像、疾病方向、科研套路、交互网络图和人工核实参考文献。

## 页面

GitHub Pages: https://jivonkiang.github.io/research-navigator-hub/

## 当前设计

- 总览界面：显示研究组合、优先级、参考文献核实进度。
- 导航栏：总览、交互网络、Idea 库、文献核实、数据质量。
- 交互网络：左侧是研究方向，右侧是方法套路，连线代表可推进 idea。
- Idea 详情：展示研究问题、可行性、研究设计、数据源、预期产出和文献证据。
- 文献核实：所有参考文献默认 `?`，人工确认后可切换为 `✓`，否决可切换为 `×`。
- 结核因果分析：作为交互分析网络中的疾病节点，不再作为孤立入口。

## 数据文件

- `data/research-data.json`: 领域、方法、idea、参考文献、论文与数据库数据。

## 维护原则

- 正式证据区不展示 `GEN...` 伪 PMID。
- 无 PMID 的文献保留 DOI 或 PubMed 标题检索入口。
- 核实状态保存在浏览器本地，后续可扩展为导出 JSON 后提交回仓库。
