Feasibility Map

研究路线图

手机端不再显示拥挤的力导向图，改成分层路线：个人基础、主线推进、投稿复盘、储备方向一眼可读。

Idea Library

Idea 库

默认只看摘要；点击卡片后再展开研究设计、数据源、产出物和文献。

Routine Evidence

套路重点文献

这里放方法套路的基础文献，不和你的 ORCID 文章混在一起。默认收起，需要时展开核实。

Reference Verification

文献核实

页面直接读取 `data/verification.json` 的自动核验结果。状态来自 PubMed 与 Crossref 的 title、PMID、DOI 交叉检查。

Data Quality

文献真实性自动核验

核验流程改为自动化：优先用 PubMed PMID 取回论文记录，用 Crossref DOI 取回出版信息，再比较 title、PMID、DOI。三项一致标记为“三重通过”，缺 PMID 但 DOI 与标题一致则标记为“部分通过”。

Submission Review

已完成但投稿不顺的课题不再占主线推荐位

多组学+MR+结核被放入“投稿复盘”。页面会保留它的证据与复盘路径，但主线会更偏向 GBD 蓝海筛题、ORCID 热点匹配和可拆分的新课题。
